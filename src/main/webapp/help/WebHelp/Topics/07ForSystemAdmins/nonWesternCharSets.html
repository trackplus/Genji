Non-western character sets

Internally Genji uses Unicode and UTF-8 for page encryption. We recommend that you use UTF-8 or UTF-16 for your e-mail templates and database connection.

The configuration of the database character encoding very much depends on the database system being used, so no general instructions can be given here. We strongly recommend to use UTF-8 or UTF-16 at the database interface as well.
