Managing the system

System manager tasks overview

This chapter deals with tasks that are usually reserved to the Genji system managers. The system manager can do anything a system administrator can do, except for configuring the server itself.

System management tasks can be categorized as follows:

  • Managing users and groups.
  • Managing cost centers and accounts.
  • Managing organizational units like companies or departments
  • Providing default cockpits
  • Importing data into and exporting data from Genji
  • Sending broadcast e-mails to Genji users
