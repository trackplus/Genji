Working with the cockpit / Configuring tiles

Activity Stream

This tile gives you for each project you have read permission in a list of the last X events, where you can configure X. This permits you to monitor project activities in real time.

Tile configuration

There is little configuration for this tile. You can set

  • the number of events to be displayed per project

  • the update interval in seconds. Set this to a reasonable value to keep server load low.
