Version Control Activity Stream

This tile permits you to see the last X commits for a number of projects, where you can configure X and the projects to consider.

Tile configuration

Before you can use this tile you need to configure it. Follow the general procedure to configure a cockpit tile outlined above to get to version control activity stream configuration page.



You need to select at least one project or release and the number of commits you want to see for each project or release.
