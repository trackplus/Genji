Working with items

The item is the central object in Genji. Genji helps to manage items. All items have a „life-cycle“. At some point in time an item is being created. People work on it, changing the item state, assigning another responsible person, adding comments, and so on. At some point in time the item can be closed.

Items have properties. When working on items you are either changing item property values or you are moving items to a different workspace.
