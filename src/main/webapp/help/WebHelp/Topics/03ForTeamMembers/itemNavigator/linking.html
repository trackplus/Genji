Linking items

You can link items directly from the item navigator.

  1. In the item navigator, mark at least two items you like to link to each other.


  2. Click on the „Action“ button and select from the menu „Link items“.
  3. Select the desired link type, eventually add parameters and save.


The items are now linked with each other. You can use this link when defining filters. For some link types the associated logic may prevent you from closing items linked to open items, or give you a temporal order like finish-start or finish-finish.
