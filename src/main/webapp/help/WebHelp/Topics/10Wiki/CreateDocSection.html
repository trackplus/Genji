Creating document sections

To create a new document go into the Wiki navigator and right mouse click on a document or document section node. Then click on "Add document section".



Creating a document section

When you are in full edit mode you can create a new document section by assigning a heading format like “Heading 1”, “Heading 2”, etc. to the section title.
