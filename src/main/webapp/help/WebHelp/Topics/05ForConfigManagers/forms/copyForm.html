Copying Custom Forms

This section describes how to copy a custom form.

You must be logged in as system manager or system administrator.

  1. In the top toolbar selectAdministration > Customize > Forms.


  2. A new screen appears. Click on the „Copy“ icon in the Action column on the left for the form you want to make a copy of.



You have created a copy of the original form. You can now edit this new form without changing the original form.
