Managing custom properties

This section describes how to add, overwrite, modify, and delete custom properties.

  • Adding a custom property
    This section describes how to add a custom property to your installation.
  • Overwriting properties
    You can create variations of properties at the workspace level, workspace type level, or item type level. By overwriting properties you can modify any of the property properties except for its type. You can for example change the label, selection lists, tool tip, or the "is required" property.
  • Modifying properties
    You can modify properties globally. For system properties, you can only change the name, label, and tool tip. For custom properties, you can change anything except for the property type.
  • Deleting custom properties
    This section describes how to permanently delete a custom property.
