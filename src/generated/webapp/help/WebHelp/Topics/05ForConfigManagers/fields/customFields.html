Customizing Track+

Item properties

Items have properties. There are several system properties predefined which you can modify, for example change their label or set default values. Beyond these pre-defined properties you can define custom properties. There are a number of property types like text properties, numeric properties, selections, and date properties provided with Track+. It is possible to extend the number of property types using the Track+ plug-in architecture.

  • Item properties overview
    Track+ has built in system properties, and you can add any number of custom properties.
  • Item property types
    Items have properties like a title or the date they were last modified. This section describes the types of item properties that are available in Track+. New item property types can be added as plug-ins, without modification of the core software.
  • Managing custom properties
    This section describes how to add, overwrite, modify, and delete custom properties.
