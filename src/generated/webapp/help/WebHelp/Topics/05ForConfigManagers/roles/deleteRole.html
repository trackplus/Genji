Deleting roles

As system administrator, you can delete roles.

You must be logged in as system manager or system administrator.

  1. In the top toolbar select Administration > Customize > User roles. Alternatively click on the "Customize"/"User roles" bar in the left navigation area.


  2. Select a role by clicking on it and the click on the "Delete" button in either the toolbar or the context menu.


You have deleted the selected role. All permission to users via this role have been revoked.

Attention: When you remove the only role of a user for a workspace, this workspace will become invisible and inaccessible to that user.
