Administering the server

Concepts and tasks for system administrators

Some operations like connecting a Track+ server to an e-mail server, providing disk space, or performing database backups and restores are reserved to Track+ system administrators. Track+ comes with a built in special user, named „admin“, initial password „tissi“.

System administration tasks can be categorized as listed below.

  • Server configuration, usually done once after system installation.
  • Setting login page text. This may be an ongoing activity to advice users of downtimes or other important information.
  • Sending broadcast e-mails to notify users of changes or planned down times.
  • Monitoring the system state and analyzing the system log.
  • Configuring logging. Required only in case of system problems for debugging purposes.
  • Performing database backups.
  • Restoring a database.
