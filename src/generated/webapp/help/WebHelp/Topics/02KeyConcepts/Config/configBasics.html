Key concepts

Configuration basics

When you deal with a lot of projects or spaces, and different kinds of projects like electronic development, software development, mechanical engineering, or controlling, configuration can become complicated. To handle large, diverse configuration scenarios, Track offers space types and configuration inheritance. Using these two features you can create diverse and flexible configuration scenarios with little effort.

  • Workspace types
    You can centralize the configuration of spaces via „workspace types“.
  • Configuration inheritance
    Configurations can be set at various levels: globally, item type specific, workspace type specific, and workspace specific.
