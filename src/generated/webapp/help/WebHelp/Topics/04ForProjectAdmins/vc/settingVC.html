Track+ Service Desk for space administrators

Version control

With Track+ Service Desk you can link files in a version control repository like Subversion, GIT or CVS to items.

Track+ Service Desk scans the repository log files on a regular basis for references to items and automatically creates links for the items. Initially, the scan takes place every 10 minutes. The scan time is measured, and the delay between the scans is set to two times the scan time. For example, if the scan time for all repositories is 20 minutes, the repositories will be scanned once each hour.

The parameters for the initial scan time and the delay factor can be set in file TRACKPLUS_HOME/quartz-jobs.xml.

  • Subversion version control
    Track+ Service Desk can integrate with your source code version control system, for example CVS, Subversion, or Perforce. To establish links, add item numbers to your version control system submits like „#<item no.>, e.g. „#4711“. This section describes for Subversion (SVN) how to proceed.
