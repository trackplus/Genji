Track+ for space administrators

Importing tasks from MS Project

As a project administrator, you can import an entire project from an MS Project database file. During the import process, Track+ tries to synchronize MS Project tasks with existing tasks within the Track+ system.

You must be logged in as system manager or system administrator.

Make sure you have saved your MS Project plan in MS Project XML format.

MS Project offers some features not available in Track+. You can use MS Project to create and maintain your project plan, do your resource planning, and later on you can use it to monitor project progress while keeping everything safe in your Track+ database.

When you import an MS Project plan into Track+, all MS Project data is preserved within Track+. Later on you can export all data back to MS Project, to make more modifications there. You can change MS Project tasks within Track+, and incorporate these changes back into your MS Project file.

Table 1. Mapping of properties between Track+ and MS Project
Track+ MS Project Hints
Planned value (work) duration * assignment units In case there is no assignment we assume hours per work day and one assigned resource
Budget (work) duration * assignment units of baseline 1 values if present, otherwise planned value (work) In case there is no assignment we assume hours per work day and one assigned resource
Actual work performed actual duration * assignment units In case there is no assignment we assume hours per work day and one assigned resource. This value is only imported when there is no entry for actual work performed in Track+.
  1. Go to the item navigator. Drag the MS Project file you want to import to the item navigator.
  2. Select the project and release into which you want to import the tasks from the MS Project file as Track+ tasks. Click on “Next”.
  3. Provide Track+ with information how to map resources in MS Project to users in Track+. You can create new users, or just map to existing ones. If you create new users, you also have to provide valid e-mail addresses.


Track+ will update existing tasks within its database, or will create new tasks with item type “task”. The entire MS Project data file is preserved withing Track+ so that information can later be merged back and exported for further processing.
