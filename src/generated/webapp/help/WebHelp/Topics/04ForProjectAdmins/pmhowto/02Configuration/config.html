Track+ for space administrators

Configuration outline

To configure TRACK for managing your first project you need to go through these steps:
  1. Create a new workspace based on one of the existing workspace types. To start, think of a workspace representing a project.
  2. Register your team members.
  3. Assign roles to your team members in your new project.

Each project is associated with a workspace type from which it inherits its configuration. By configuring workspace types you configure all workspace of that type in one step, even later when the projects are already underway.
