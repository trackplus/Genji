Track+ for space administrators

Exporting a project to MS Project

As a project administrator, you can export an entire project into an MS Project database file. You can modify the data in MS Project, and later on import the file back into Track+.

You must be logged in as space administrator, system manager or system administrator.

MS Project and Project Libre offer some features not available in Track+. You can use MS Project or Project Libre to create and maintain your project plan, and later on you can use it to monitor project progress while keeping everything safe in your Track+ database.

You can export an entire Track+ workspace or just a specific release into an MS Project database file. The file format is MS Project XML. You can open the file directly from MS Project for viewing and analyzing.

  1. In the item navigator list all items you want to export to an MS Project file and select them using the check boxes on the left or using your mouse (with shift or command key held down).
  2. Go to Actions > Export > Export to MS Projectand export the selected items to an MS Project file.


A download dialog will open and you can download the MS Project XML file to your local computer. Here you can open the file with Project Libre or MS Project for analysis and modifications.
