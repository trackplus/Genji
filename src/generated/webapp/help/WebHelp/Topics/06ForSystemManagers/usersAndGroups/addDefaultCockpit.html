Adding a cockpit template

You can add a new cockpit template from scratch or you copy an existing one. Two create a new cockpit template, you first give it a name and description, and then you configure it.

You must be logged in as system manager or system administrator.

  1. In the top toolbar selectAdministration > Users and groups > Default cockpit. Alternatively click on the "Users and groups"/"Default cockpit" bar in the left navigation area.


  2. In the small toolbar, click on "Add" and give the new cockpit template a name and optionally a description.


  3. Save the new template definition by clicking on the "Save" button.
  4. Proceed to Configuring a cockpit template
