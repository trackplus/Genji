Exporting space type configurations

You can export complete workspace type configurations including custom properties and screens from your Track+ instance and later import it into a different instance. This permits you to develop a Track+ configuration on one machine and transfer it to a production machine when it is ready.

You must be logged in as system manager or system administrator.

  1. Go to Administration > Customize > Workspace types
  2. Select a workspace type and click on the Exportbutton.
You will get an XML file containing the entire workspace configuration. This includes
  • Item types and item type assignments
  • Forms and form assignments
  • Item properties and fields
  • Specific lists and options
