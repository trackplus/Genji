Exporting to an Excel file

You can export the items shown in the item navigator to an Excel spread sheet file.

  1. Click on the „Action“ button and select from the menu „Excel“.
  2. Download the Excel file to your local computer. You can directly open this file in Excel or OpenOffice.

The Excel file contains all current properties of the items shown in the item navigator. The Excel file does not contain any trail information, due to the limitations of the format. If you have selected items in the item navigator, only those items will be exported.
