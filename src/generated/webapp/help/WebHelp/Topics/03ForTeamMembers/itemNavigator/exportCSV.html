Exporting to a CSV file

You can export the items shown in the item navigator to a CSV formatted file for import into applications that can read this format (e.g., spread sheets).

Note: You can set the separator character which is being used in the CSV file in your personal profile area, since it may depend on your version of MS Office and the locale on your client computer.
  1. Click on the „Action“ button and select from the menu „CSV“.
  2. Download the CSV file to your local computer. You can directly open this file in Excel or OpenOffice.

The CSV file contains all current properties of the items shown in the item navigator. The CSV file does not contain any trail information, due to the limitations of the format.
