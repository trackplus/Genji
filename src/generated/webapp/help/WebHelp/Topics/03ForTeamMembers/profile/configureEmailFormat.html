Configuring e-mail format

You can select if you want to receive e-mails in plain text format or in HTML format. The HTML format looks much better and quickly gives you an overview on what has changed. Most e-mail clients support the HTML format well, so there may be little reason to not choose this.

  1. In the top toolbar select Administration > My profile > My settings.


  2. Select the tab labeled "Reminder e-mails".


  3. Select the desired e-mail format, „plain“ or „HTML“.
  4. Click on the "Save" button in the tool bar to save your configuration.

From now on, Track+ will send all notification e-mails to you using the selected e-mail format.
