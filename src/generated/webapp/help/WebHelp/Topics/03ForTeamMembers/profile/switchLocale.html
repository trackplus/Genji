Configuring the locale

The Track+ system has been localized in many different languages. You can choose which language is presented to you by either setting the preferred language in your browser, or by setting the language in your profile.

  1. In the top toolbar select Administration > My profile > My settings.


  2. Select the tab labeled "Access data".


  3. Choose your preferred user interface language settings. „Browser“ means that your browsers preferred language settings are being used, if that language is available.
  4. Click on the "Save" button in the tool bar to save your configuration.

You will now see the user interface in the selected language. If set to „Browser“ your preferred language setting may not be available. Furthermore, you will receive notification e-mails in the servers default language.
