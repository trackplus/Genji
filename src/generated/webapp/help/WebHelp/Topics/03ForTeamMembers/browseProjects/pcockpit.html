Track+ for team members

Browsing projects

You can have all important information for a specific project pulled together in a project specific cockpit. You can select which project you want to see using a project and release navigator. The cockpit is configured by the project administrator so that each project member has the same view.

Other than that the cockpit has a navigation area on the left, and that it cannot be configured by regular users, the functionality is the same as that of the user specific cockpit. All links and information shown is automatically restricted to the chosen project and phase.
