System properties

The table below describes all system properties Track+ knows about. In addition, you can define any number of custom properties. The labels can be localized and may have been changed on your system. When you change the property name you have to use that new name in all places like report templates and TQL filters.

Table 1. System properties reference

ArchiveLevel   „deleted“ or „archived“
